HoverTransitions v1.0 - Help Page

How to use

JavaScript

You must include the following jQuery code in the head element.

<script type="text/javascript" src="scripts/jquery-1.7.2.min.js" ></script> <script type="text/javascript" src="scripts/jquery-ui-1.8.22.custom.min.js" ></script> <script type="text/javascript" src="scripts/jquery.hover-transitions-1.0.min.js" ></script>

The jQuery UI custom code includes the slide and scale effects and the effects core. These are required

HTML

The plugin is designed to work with all HMTL content. Create the element that you want to animate and add the relevant CSS style to it. The element can have an ID or a class name so that it can be accessed by the jQuery plugin.

<a id="p1" class ="button" href="http://codecanyon.net">Press </a>

CSS

The following is an example of the CSS code used to style the element:

.button { font-weight:normal; font-size: 20px; background-color: #2EA6FF; color: #fff; border-radius: 25px; padding: 30px; }

Those will be the style properties for the initial appearance of the element; some of these properties can also be set via the HoverTransitions plugin properties code. The style of the object after the animation can be defined by creating another class for it or directly via the plugin properties code.

Running the plugin

The following JavaScript code is used to start the plugin. Multiple instances of the plugin can be executed on each page.

$('#p1').hover_transitions({ innerHTML_two : "Me", background_color_two: "#ff286c", pattern : "spiral" });

Please note that the last property must not have a trailing comma in order for it to work in Internet Explorer 7.

How the animation works

The target element is not animated; it remains untouched. Instead an invisible clone of the element is created (with all the required changes) and it is made to appear over the target element.

Plugin properties

Below is a list of all the possible properties and an explanation of what they do. To understand some of the properties, please look at the index page for examples.

Properties with default values:
PropertyDefault ValueDescription
pattern "diagonal" Animation pattern. Possible values: "diagonal", "regular", "spiral", "crossing", "corners_alt", "skip_rows", "skip_columns", "checks", "zip", "random", "corners" and "wave".
flip_pattern_horizontal false Flip the patterns horizontally.
flip_pattern_vertical false Flip the patterns horizontally.
rotate_pattern false Rotate the pattern by 90 degrees.
pattern_reflection 1 Reflect a pattern. Accepts values 1, 2 and 4. Only works with "diagonal", "crossing", "zip" and "regular" patterns.
show_method "fade" The method used to male each box appear. Possible values: "fly", "fade", "puff", "grow", "sliding_doors_horizontal", "sliding_doors_vertical", "alt_slide_vrtical", "alt_slide_horizontal", "slide_up", "slide_down", "slide_left", "slide_right", and "instant".
hide_method "fade" The method used to hide each box. Accepts the same values as the show method. The show and hide methods do not need to be the same.
fly_show_direction "random" The flight path of the boxes when the fly method is used. Possible values: "random", "up", "down", "left", "right", "split_vertical", "split_horizontal", "alt_vertical", and "alt_horizontal".
fly_hide_direction "random" Accepts the same values above but the hide and show values do not need to be the same.
fly_hide_fade_speed 300 Duration in milliseconds. The amount of time taken for the boxes to fade away once they have reached their destination.
fly_distance 100 The flight distance in pixels away from the target element.
rows 6 The number of rows in the pattern.
cols 6 The number of columns in the pattern.
pattern_speed 30 The time interval (in milliseconds) between the animation of each box in a pattern.
show_speed 500 The time in milliseconds taken for the show effect to be completed for each box.
hide_speed 500 The time in milliseconds taken for the hide effect to be completed for each box.
event_mode "hover" Accepts values "hover", "click" and "none". The event method with which the animation will be controlled. Use "none" if you donâ€™t want the element to react to any events (e.g. if another element is controlling the animation of this element).
rewind_on_mouse_out true Specifies if the animation can be reversed or not.

Properties without default values:
PropertyExample ValueDescription
alt_hover_control "#alt_hover" The jQuery selector of the alternative element that will control the hover event.
alt_click_control_one "#alt_click_one" The jQuery selector of the alternative element that will control the show animation when clicked. If the alt_click_control_two property is not set then this will control the hide animation on toggle.

Any specific properties set directly via the plugin will override the properties set using CSS style sheets. Specifically set properties will also override the ones set in classOne and classTwo.
